## Мультиустройство и ключи

- Каждое устройство получает **свои wrapped keys**; история backfill до 90 дней при привязке.
- Потеря устройства: отозвать в списке сессий → wrapped keys для него не выдаются.
- См. [key-lifecycle.md](../03-security/key-lifecycle.md), [sync-offline.md](../04-data/sync-offline.md).
