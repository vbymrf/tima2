# Окно комментариев

> Оверлей (телефон/планшет) или правая панель (десктоп, режим слайдов).
> **Комментарии — общая подсистема** ([module-boundaries.md](../02-architecture/module-boundaries.md) §2): один UI и API для постов каналов, публичных медиа и публичных коллекций. Только публичный контур — в защищённых чатах обсуждение = сообщения и ветки.

## Верхняя панель (корневой уровень)

::: row
[[ **Комментарии (8)** ]]
[[ ✕ ]]
:::

> Только **✕** справа. **Назад** — при входе во **вложенную ветку** (слева).

## Сводка эмоций

::: row
👍 12  😂 5  🔍 3  😖 1
:::

> Только эмоции канонической шкалы 9 ([18-content-actions.md](./18-content-actions.md)).

## Список комментариев

::: row {left}
:user: **UserA**
Текст комментария 1… `12:30`
:::

::: row
[Ответить] [😊] [↗]
:::

::: row {left}
  :user: **UserB** (ответ на A)
  Текст ответа… `12:35`
:::

::: row
  [Ответить] [😊]
:::

::: row {left}
:user: **UserC**
Текст комментария 2… `12:40`
:::

> **Вложенность:** один визуальный уровень отступа; более глубокие ответы — в той же ветке с @username.

## Поле ввода

::: row
[[ Написать комментарий... ]]
[[ ➤ ]]
:::

> При [Ответить] — поле подхватывает цитату с @username.

## Вложенная ветка

::: row
[[ [<] Назад ]]
[[ **Ответы на UserA** ]]
[[ ✕ ]]
:::

> Esc (десктоп) — закрыть оверлей.

## Адаптация

> **Десктоп (слайды):** слева контент, справа комментарии — без полноэкранного оверлея. См. [09-media-slides.md](./09-media-slides.md).
