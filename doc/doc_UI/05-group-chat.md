# Групповой чат

> **Типы:** личная группа (E2E) / публичная группа (без E2E). **Упоминания:** @username.

## ЗОНА 1 — Верхняя панель

::: row
[[ [<] ]]
[[ :user: ]]
[[ **Команда разработки** `(3 участника)` ]]
[[ :phone: ]]
[[ :video: ]]
[[ :more: ]]
:::

> **:phone: / :video:** — только если группа поддерживает звонки/аудиочат. Иначе скрыты.
> Клик по аватару/названию → [16-profile-popup.md](./16-profile-popup.md).

## Строка эмодзи + ЗОНА 4

::: row
😀 😂 👍 ❤️ …
:::

::: row
[[ :emoji: ]]
[[ Напишите сообщение... @ для упоминания ]]
[[ :image: ]]
[[ :microphone: ]]
:::

> При вводе `@` — всплывающий список участников над полем. Упомянутый — отдельный push.

> **Без права отправки:** скрыты строка эмодзи и ЗОНА 4 (режим «только чтение»).

## Публичная группа — оценки

::: row
[+] [−] [😊] [↗]
:::

> Только для **публичного** контента. В E2E-группах [+]/[−] скрыты.
