## Избранное — две полки

> Сквозные закладки из окон 2 и 3 (⭐, при сохранении — выбор полки одним тапом). Модель: [feed-ranking.md](../04-data/feed-ranking.md) §2.

::: row
[Публичная полка]* [Личная полка 🔒]
:::

**Публичная полка** — видна друзьям на странице, питает их ленту друзей:

::: card
:user: **Пост из медиа-ленты** — добавлен `вчера`
[Открыть] [→ В личную] [Удалить]
:::

**Личная полка** — шифруется (shelf_key), в ленты не попадает никогда; расшифровка на устройстве:

::: card
🔒 **Закладка** — добавлена `вчера`
[Открыть] [→ В публичную] [Удалить]
:::

> **Доступ к личной полке по запросу:** входящие запросы — бейдж на вкладке; [Разрешить] = передача ключей (wrapped shelf_key на устройства друга), [Отозвать] = ротация ключа. Чужая личная полка видна на странице пользователя только после разрешения.
