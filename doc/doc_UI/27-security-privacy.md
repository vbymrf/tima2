# Безопасность и конфиденциальность

> **Правило UI:** статус «Защищённый чат» / «не защищён» — **только** в окне информации [16-profile-popup.md](./16-profile-popup.md), не в шапках лент.
> **Терминология:** в UI «E2E» = шифрование на устройстве; технически — **controlled escrow** ([escrow-legal-access.md](../03-security/escrow-legal-access.md)). В настройках — ссылка «Политика конфиденциальности».

## Матрица шифрования

| Тип контента | Защита (UI) | Примечание |
|--------------|-------------|------------|
| Личные чаты | Да (E2E) | + escrow blob |
| Личные группы | Да (E2E) | Sender Keys |
| Личные коллекции | Да (E2E) | |
| Публичные группы, каналы | Нет | Plaintext на сервере |
| Голосовые сообщения (async) | Да (E2E) | Kodium → MinIO |
| Голосовые чаты, звонки | Транспортное (SRTP) | LiveKit, не app-E2E |

## Архив и история

- Архив чатов — скрыть из списка
- Очистка: для себя / для всех (**«для всех»** = soft delete; escrow retention по политике)
- Автоудаление через N времени
- Экспорт PDF / HTML / TXT (расшифровка на клиенте)
