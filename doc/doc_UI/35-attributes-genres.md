# Атрибуты и жанры

> **Атрибуты = хэштеги** (единый реестр, создают авторы). **Жанр** — курируемая сервером «папка» атрибутов. Модель: [feed-ranking.md](../04-data/feed-ranking.md) §3. Тап по `#атрибуту` в любом посте открывает карточку атрибута.

## Чипсы над общей лентой (окна 2 и 3)

::: row
[Для вас]* [Наука и технологии] [#программирование] [#kotlin] [Политика] [+ Ещё]
:::

> **[Для вас]** — скоринг по оценкам без тематического среза. Жанры — из добавленных атрибутов (их жанры подтягиваются автоматически). **[+ Ещё]** → управление «моими атрибутами».
> Тап по жанру → срез по всем атрибутам жанра; тап по атрибуту — только по нему.

## Создание атрибута (только из редактора)

> Атрибуты создаются **авторами при публикации** ([34-content-editor](./34-content-editor.md)): поле атрибутов с автодополнением; если совпадений нет → «Создать #новый-атрибут?» → выбор **предлагаемого жанра** из списка. Сервер может принять, переназначить жанр или вернуть рекомендацию «похожий атрибут уже существует».

## Состояния

> **Атрибут без жанра** (ещё не классифицирован): карточка без строки жанра; в чипсы попадает как отдельный атрибут.
> **Пост с неодобренной меткой** (`declared`): в тематическом срезе не показывается; у автора — пометка «метка на проверке».
