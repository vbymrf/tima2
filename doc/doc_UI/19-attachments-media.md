## Голосовые сообщения

> Запись: **удержание** :microphone: или переключение после ввода текста.
> **Плеер:** 1.5x / 2x, перемотка ±10 сек.
> **Статус исходящего:** ✓ доставлено → ✓✓ прослушано (для голосовых «прослушано» заменяет «прочитано» в трёхуровневой модели доставлено/прочитано/прослушано).

### Pipeline (E2E чаты / private коллекции)

1. **Клиент:** Opus/JPEG/WebP → zstd (опционально) → **Kodium SecretBox** (chunked если >10 МБ).
2. **Upload:** presigned URL → MinIO (**только ciphertext**).
3. **Сообщение:** указатель `media_ref` + wrapped `media_key` на сервере.

> Публичные чаты: plaintext upload в отдельный bucket. См. [media-storage.md](../04-data/media-storage.md).

### Pipeline (публичные чаты)

Upload → MinIO plaintext → сообщение со ссылкой (без Kodium).
