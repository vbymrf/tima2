> **Исходящие:** справа, без аватара. Статусы: индикатор → ✓ доставлено → ✓✓ прочитано → ! ошибка (повтор).

> **ЗОНА 3:** [↑] и [+5] — всегда активны. [↻] [😊] [↗] — активны **после выбора сообщения**. [↻] = ответ с цитатой; [↗] = переслать. См. [18-content-actions.md](./18-content-actions.md).

## Строка часто используемых эмодзи

::: row
😀 😂 👍 ❤️ 🔥 🎉 👋 🙏 …
:::

> Видна **всегда**, если есть право отправки. Горизонтальный свайп — **только прокрутка эмодзи**, не переключение окон. Вставка в поле ввода.

## ЗОНА 4 — Отправка (60px, на всю ширину)

::: row
[[ :emoji: ]]
[[ Текст сообщения... ]]
[[ :image: ]]
[[ :microphone: ]]
:::

> **:image:** → меню вложений (фото, видео, файл, камера). См. [19-attachments-media.md](./19-attachments-media.md).
> **:microphone:** → запись голосового (удержание). При **≥1 символе** в поле — заменяется на [➤ Отправить]*.

> **Двойной тап** по 😊 без выбора → палитра для **последнего сообщения другого пользователя**. Одинарный тап без выбора — без действия.

## Состояния без отправки

> Если отправка запрещена: скрыты строка эмодзи и ЗОНА 4. Правая панель и чтение — доступны.

## Адаптация

> **Планшет:** кнопки ЗОНЫ 3 — горизонтальный ряд под полем ввода: [↑] [↻] [😊] [↗] [+5].
> **Десктоп:** ЗОНА 3 — тулбар под верхней панелью; [+5] дублируется плавающей справа внизу. См. [31-adaptive-layout.md](./31-adaptive-layout.md).
