# Голосовой чат / аудио-чат

> **Привязка:** аудио-чат существует **только внутри сообщества** (без сообщества не создаётся; «отдельное» создание автоматически создаёт сообщество — [communities.md](../01-product/communities.md) §2). Дополнительно может быть прикреплён к группе/каналу того же сообщества. **E2E:** нет. Может стать **окном 0** при активном участии.

## Точка входа

> Из [05-group-chat.md](./05-group-chat.md) / [06-channel.md](./06-channel.md) (вложенный) или со страницы сообщества [32-community.md](./32-community.md) (самостоятельный). Создание → [33-create-group-channel.md](./33-create-group-channel.md).
