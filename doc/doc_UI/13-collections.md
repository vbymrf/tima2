## Просмотр элемента (полноэкран)

> Тап по превью в сетке открывает просмотр.

::: card
**ФОТО / ВИДЕО**
:::

::: row {center}
[◄] ● ○ ○ [►]
:::

> При **>1** элемента в просмотре — переключение **только** [◄] [►] и точками. Свайп запрещён. См. [01-app-shell.md](./01-app-shell.md).

> **Совместная коллекция** — приглашение друзей на наполнение. Уровень приватности в [27-security-privacy.md](./27-security-privacy.md).
