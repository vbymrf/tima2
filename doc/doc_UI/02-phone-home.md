## Состояния

> **Empty state:** «Нет чатов» + [Найти контакт]*

> **Loading state:** скелетон списка.

> **Offline:** баннер «Нет сети», история из кеша.

## Адаптация

> **Планшет/десктоп:** список чатов слева (30–35%), выбранный чат справа. См. [31-adaptive-layout.md](./31-adaptive-layout.md).
