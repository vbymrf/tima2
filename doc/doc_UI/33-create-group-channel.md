> Аудио-чат **без сообщества не создаётся**. «Создать новое» → авто-сообщество (имя = имя аудио-чата, можно переименовать). Создание из группы/канала: если они в сообществе — аудио-чат попадает в то же сообщество с привязкой отображения (`attached_type/attached_id`); если группа standalone — мастер предлагает создать сообщество, включающее группу и аудио-чат. См. [communities.md](../01-product/communities.md) §2.
> Вложить сообщество в сообщество нельзя.

## Шаг 4 — Доступ и модерация

## Завершение

::: alert success
Создано. Создатель — owner. [Открыть]* [Пригласить ещё] [Настроить]
:::

> Создатель — owner элемента; при авто-сообществе — owner обоих. Роли далее: [32-community.md](./32-community.md) (сообщество), настройки группы/канала (элемент).
