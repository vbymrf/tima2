## Логика кнопок

| Кнопка | Видимость | Действие |
|--------|-----------|----------|
| Перейти на страницу | Всегда | Полный профиль / страница группы |
| Закреплённые | Если есть | Список закреплённых |
| Поиск | Всегда | Поиск в текущем чате/канале |
| Сообщения от User | Группы >2 | Фильтр по автору |
| Общие медиа | Если есть файлы | Галерея по дате |
| Уведомления | Всегда | Звук / только @ / выкл |
| Покинуть / Отписаться | Группа / канал | С подтверждением |
| Уведомления / Сообщения | Все сущности | Независимые тумблеры («и/или»): mute уведомлений; запрет карточек-сообщений от сущности в окно 4 (`block_messages`) |
| Заблокировать | Пользователь | Чёрный список |

## Поведение

> **Телефон/планшет:** модальное окно (~90% / 60% ширины).
> **Десктоп:** **правая информационная панель** (заменяет содержимое колонки «Инфо»), не центральный modal.
> Закрытие: ✕, клик вне (mobile), Esc (desktop). Состояние сохраняется при повторном открытии.
