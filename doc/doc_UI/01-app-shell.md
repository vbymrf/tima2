# Оболочка приложения

> **Назначение:** единая сессия, переключение между базовыми окнами, сохранение состояния.

## Порядок окон в горизонтальном переключателе

1. **Окно 0** — временное, только при активном звонке / видеозвонке / аудиочате. Всегда **первое** (слева). Удаляется после завершения сеанса.
2. **Окно 1** — Телефон
3. **Окно 2** — Новостная лента
4. **Окно 3** — Медиа-лента
5. **Окно 4** — Социальное взаимодействие
6. **Окно 5** — Личная страница
7. **Окно 6** — Медиа-блогер (если активирован)
8. **Окно 7** — Новостной блогер (если активирован)

> **Правило:** одновременно только **один** активный звонок/аудиочат. Новый сеанс требует завершить текущий.

## Горизонтальный свайп

Горизонтальный свайп работает в области **выше нижней панели** и делает только одно — **переключение окон** по порядку (0 → 1 → 2 → …).

**Единственное исключение** — **истории**: внутри просмотра историй свайп переключает истории автора. См. [11-stories.md](./11-stories.md).

**Нижняя панель** (поле ввода, строка эмодзи) окна не переключает: прокрутка строки эмодзи — локальный жест внутри панели. Зона вложенных медиа — только кнопки ◄ ►, свайп там окна тоже не переключает.

> Никаких свайп-действий над элементами списков (архив, удаление и т.п.) в приложении нет — такие действия только через long-press / меню.

- Состояние каждого окна (вкладка, прокрутка, выбранный чат) **сохраняется** при переключении.

## Вложенный контент (глобально)

> Переключение вложений (несколько фото/видео в одном посте, альбом в чате, карусель в ленте) — **только кнопками** [◄] [►] и точками. **Свайп запрещён** (надёжность, без конфликта с окнами).

| Экран | Правило |
|-------|---------|
| Медиа-лента «Лента» | [08-media-window.md](./08-media-window.md) |
| Медиа-лента «Слайды» | [09-media-slides.md](./09-media-slides.md) — вертикальный свайп только между **постами** |
| Чат (альбом) | [19-attachments-media.md](./19-attachments-media.md) |
| Коллекции (просмотр) | [13-collections.md](./13-collections.md) |
| Истории | **Исключение** — горизонтальный свайп между историями сохраняется |

Кнопки ◄ ► видны **только при вложений > 1**.

## Глобальный поиск

[[ :search: Поиск... ]]

> На десктопе — по центру верхней панели. Шорткат: `Ctrl+K` / `Cmd+K`. См. [17-global-search.md](./17-global-search.md).

## Клавиатурные шорткаты (десктоп)

|----------|---------|

| Отправить сообщение | Enter |
| Новая строка | Shift+Enter |
| Переключить чат ↑/↓ | ↑ / ↓ |
| Настройки чата | Ctrl+, |
| Видеозвонок | Ctrl+Shift+C |
