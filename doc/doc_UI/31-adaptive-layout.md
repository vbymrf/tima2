# Адаптивная вёрстка

> **Стек:** Compose Multiplatform. **Breakpoints (dp):** Compact <600, Medium 600–839, Expanded ≥840.

## Матрица жестов (канон)

| Жест | Назначение |
|------|------------|
| Горизонтальный свайп (основная область) | Переключение окон 0–7 |
| Горизонтальный свайп (строка эмодзи) | Прокрутка эмодзи |
| Горизонтальный свайп (Истории) | След./пред. история автора — исключение |
| Горизонтальный свайп (вложения в посте) | **Запрещён** |
| Вертикальный свайп (Слайды) | След./пред. **пост** |
| [◄] [►] и точки | След./пред. **вложение** (>1 в посте/альбоме) |

> Подробнее: [01-app-shell.md](./01-app-shell.md), [08-media-window.md](./08-media-window.md), [09-media-slides.md](./09-media-slides.md).

## Телефон (Compact)

| Элемент | Поведение |
|---------|-----------|
| Навигация окон | Горизонтальный свайп по основной области |
| Вложенные экраны | [<] сверху слева |
| Личный чат | columns-2: сообщения + правая панель управления |
| Нижний редактор | На всю ширину под messages |
| Окно переходов | Modal ~90% ширины |

## Планшет (Medium)

| Элемент | Поведение |
|---------|-----------|
| Верхняя панель | Высота 64px (тач-цели) |
| Чаты | 30–35% список \| 65–70% текущий чат |
| ЗОНА 3 (чат) | Горизонтальный ряд под полем: [↑] [↻] [😊] [↗] [+5] |
| Медиа-лента | Сетка 2–3 колонки |
| Слайды | Контент ≤80% ширины, кнопки дублируются снизу |
| Окно переходов | Modal ~60% ширины |

## WireMD-ограничения

> Точные px/dp и fixed/floating позиции в макетах **приблизительны**. Канон — текстовые правила в этом файле и в экранных ТЗ.
> Использовать только `columns-N`, `column`, `row`, `card`, `modal`. **Не** использовать `grid-*`, HTML, CSS.
