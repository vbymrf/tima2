## Переключение вложений

> **Несколько фото/видео в одном посте** (вложений > 1):

- Переключение — **только** кнопками [◄] [►] и тапом по точкам ● ○ ○.
- Горизонтальный свайп по карточке медиа **не** переключает вложения и **не** переключает окна приложения.
- Блок [◄] ● ○ ○ [►] **скрыт**, если вложение одно.

> **Одно видео в посте** (как UserY ниже): стрелки и точки скрыты. Тап по превью ▶ 3:45 — воспроизведение в карточке; перемотка — прогресс-бар плеера. См. [19-attachments-media.md](./19-attachments-media.md).

> Глобальное правило жестов: [01-app-shell.md](./01-app-shell.md#вложенный-контент-глобально).
