# Панель действий над контентом

> **Универсальный редактор:** виден при отображении контента (текст, медиа, комментарий). Разделение: **оценка контента** ≠ **отправка эмодзи-сообщения**.

## Две зоны

| Зона | Когда | Элементы |
|------|-------|----------|
| Панель действий | Чтение / выбор сообщения | [+] [−] [😊] [↻] [↗] [📌] [🚩] |
| Редактор сообщений | Право отправки | строка эмодзи + поле ввода |

> Без права отправки — только панель действий (где применимо).

## [+] / [−] — рекомендация (публичный контент)

> **Только** публичные группы, каналы, медиа-лента. **Не** в личных E2E-чатах.
> Взаимоисключающие: лайк / дизлайк для алгоритма общей ленты.
> **Удаление** — через меню (🗑), не через [−].

## [😊] — шкала эмоций (оценка сообщения)

Шкала из **9 эмоций**: 4 пары с валентностью ±1 + одна нейтральная.

| Шкала | +1 | −1 |
|-------|----|----|
| Статус | 👍 Одобрение | 🙄 Презрение |
| Разрядка | 😂 Смех | 😖 Боль |
| Угроза | 👊 Ярость | 😨 Страх |
| Внимание | 🔍 Интерес | 😑 Скука |
| Покой | 🧘 Спокойствие — нейтральная (0) | |

**Правила:**

- Одна эмоция из шкалы на сообщение; записывается в БД как элемент сообщения.
- Под сообщением выводятся иконки эмоций со счётчиками (агрегат по всем оценившим).
- **Метрика рейтинга:** 8 эмоций дают +1 или −1; засчитывается в **раздельные счётчики** положительных и отрицательных оценок по автору контента и по группе/каналу (например, «+33 −22»). Счётчики **не суммируются** в одно число. 🧘 Спокойствие в метрику **не записывает ничего**.
- Рейтинг показывается в профиле/окне информации ([16-profile-popup.md](./16-profile-popup.md)).
- Шкала эмоций ≠ [+]/[−]: рекомендации управляют выдачей общей ленты (только публичное), эмоции — оценка и рейтинг (любой контент).
- **Одинарный тап** [😊]: без выбора сообщения — **нет действия**; с выбором — палитра 9 эмоций.
- **Двойной тап** [😊]: палитра для **последнего сообщения другого пользователя** (эмоция после выбора в палитре).
- Произвольных эмодзи-реакций нет; эмодзи-строка редактора — отправка сообщений, не оценка.

> Строка над полем ввода — **отправка как сообщение**, не оценка. Горизонтальный свайп по строке не переключает окна приложения.
