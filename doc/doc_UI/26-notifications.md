# Уведомления (3 уровня)

## Уровень 1 — Глобально

::: card

Режим
( ) Все звуки
(*) Только важные
( ) Без звука

Ночной режим (тихие часы)
[22:00 — 08:00]{switch checked}

:::

> Не более **1 уведомления в 5 минут** из одного чата (умная группировка).
