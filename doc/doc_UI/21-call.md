# Звонки — все состояния

> **Окно 0:** временное, **первое** в горизонтальном переключателе. **Один** активный сеанс.
> **E2E звонков (app-level):** нет — медиа через LiveKit/WebRTC (SRTP). Статус в [16-profile-popup.md](./16-profile-popup.md). См. [ADR-0006](../adr/0006-livekit-media-policy.md), [recording-policy.md](../06-realtime/recording-policy.md).
> **Запись звонков: отсутствует** — LiveKit Egress не разворачивается; кнопки записи в UI нет ([recording-policy.md](../06-realtime/recording-policy.md)).
