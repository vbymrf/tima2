**ФОТО / ВИДЕО (весь экран)**

**ЗОНА СЕРЕДИНА** — тап → пауза/плей (видео)

::: row {center}

:::

**ЗОНА НИЗ** — прогресс-бар видео

:::

> **Вложения (>1):** переключение **только** кнопками [◄] [►] и тапом по точкам. Блок скрыт при одном вложении.
> **Горизонтальный свайп** в зоне медиа **не** переключает вложения (как в [08-media-window.md](./08-media-window.md)).
> Горизонтальный свайп по основной области — только **окна приложения**. См. [01-app-shell.md](./01-app-shell.md).
