# Глобальный поиск

> Единая строка: контакты, группы, каналы, сообщения, медиа, коллекции.

> **Private E2E сообщения:** поиск только по **локальному индексу** на устройстве (SQLite FTS). Сервер не индексирует plaintext. Public — серверный поиск (MVP: PostgreSQL FTS → OpenSearch по триггеру роста). См. [search-indexing.md](../04-data/search-indexing.md), [ADR-0007](../adr/0007-search-split.md).
> **Атрибуты** — отдельный тип результата: карточка с жанром и счётчиками → [Добавить себе] ([35-attributes-genres](./35-attributes-genres.md)).
